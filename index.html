วิดีโอ → QR

อัปโหลดไฟล์วิดีโอเข้า GitHub repo โดยตรงจากเบราว์เซอร์ แล้วสร้าง QR โค้ดที่ชี้ไปยังลิงก์ไฟล์นั้น ทุกอย่างทำงานฝั่ง client — ไม่มีเซิร์ฟเวอร์กลาง

Token จะถูกส่งตรงไปยัง GitHub เท่านั้น ไม่ผ่านเซิร์ฟเวอร์อื่นใด แนะนำให้สร้าง Fine-grained Personal Access Token ที่จำกัดสิทธิ์เฉพาะ repo นี้ และให้สิทธิ์แค่ Contents: Read and write แล้วลบทิ้งหลังใช้งานเสร็จ

01 · repository

Repo ต้องเป็น public ไม่งั้นลิงก์ raw จะเปิดดูไม่ได้ถ้าไม่ล็อกอิน (private repo ใช้ได้เฉพาะคนที่มีสิทธิ์เข้าถึง)

02 · ไฟล์วิดีโอ
